# Histogram Analysis Skill

A histogram should answer: "How is this metric distributed?"

Good solar histogram examples:
- signed contract to PTO cycle time
- days in permitting
- days from install to PTO
- project cycle-time distribution
- system size distribution
- crew install duration
- inspection failure count by AHJ

A good histogram card includes:
- clear title
- x-axis bucket labels
- y-axis counts
- meaningful bucket ranges
- benchmark or threshold if useful
- plain-English explanation of the tail risk

Do not create a histogram without explaining what the distribution means.
