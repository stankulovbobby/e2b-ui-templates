# Responsive B2B UI Skill

Design for a polished business-decision-maker demo.

Guidelines:
- clear hierarchy above the fold
- rounded white cards on a soft neutral background
- readable tables and compact status chips
- responsive grids that collapse cleanly
- no cramped mobile layout
- avoid novelty UI that distracts from the business question

The first 10 seconds should make the value obvious.
