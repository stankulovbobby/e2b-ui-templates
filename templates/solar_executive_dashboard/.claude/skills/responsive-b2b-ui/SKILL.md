# Responsive B2B UI Skill

The interface should look polished and credible in a sales demo.

Priorities:

- Strong above-the-fold hierarchy.
- Clear section titles and subtitles.
- Dense but readable business information.
- Comfortable spacing, rounded cards, subtle borders, and realistic shadows.
- Responsive behavior for desktop, tablet, and narrow screens.
- Avoid tiny illegible charts or tables on smaller screens.
- Use business-friendly language.

The app should feel like a modern operations platform, not a generic starter template.
